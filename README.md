# The Sovereignpreneur — A Declaration of Production

The source of [createownscale.com](https://createownscale.com).

A declaration of language and stance on sovereignty over money, production, and lifestyle in the age of AI. **Four axioms, one category, one protocol.**

This is a **living document**. The axioms are fixed. The language around them will sharpen over time — that's what this repository is for.

## Preview locally

The site is a single static HTML file with an `images/` directory. No build step.

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000.

## Contributing

Genuine edits, clarifications, and refinements are welcome. Please read [CONTRIBUTING.md](CONTRIBUTING.md) before opening a PR.

Short version:

- **Typos, clarifying edits, broken links** — open a PR directly.
- **Substantive revisions** (changing a claim, reframing a section, adding a new source) — open an Issue first so we can talk about it before you spend the time writing.
- **New axioms, ideological rewrites, or changes to the four fixed axioms** — will be closed. The axioms are load-bearing; the language around them is what evolves.

Every merged PR auto-deploys to createownscale.com.
